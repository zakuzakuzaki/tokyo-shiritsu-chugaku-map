# 東京私立中学校マップ (Tokyo Private Middle School Map)

東京都内の私立中学校を地図上に表示し、検索・フィルタリング機能を持つWebアプリケーションです。

## 🎯 プロジェクト概要

このアプリケーションは、東京都内の主要な私立中学校の情報を地図上に可視化し、受験生や保護者が学校選びの参考にできるツールです。学校の基本情報、偏差値、学費などの条件で検索・フィルタリングが可能です。

## ✨ 主な機能

### 必須機能
- **地図表示**: 東京都を中心とした地図上に私立中学校の位置をマーカーで表示
- **学校詳細情報**: マーカークリックで学校詳細をポップアップ・カード表示
- **検索機能**: 学校名での検索
- **フィルタリング機能**:
  - 区域での絞り込み
  - 共学/男子校/女子校での絞り込み
  - 偏差値範囲での絞り込み
  - 学費範囲での絞り込み
- **レスポンシブデザイン**: PC、タブレット、スマートフォン対応

### 表示される学校情報
- 学校名
- 住所・所在区
- 最寄り駅・徒歩時間
- 偏差値（概算）
- 共学/男子校/女子校の区分
- 年間学費（概算）
- 創立年
- 生徒数
- 学校の特色・特徴
- 学校概要

## 🛠 使用技術

- **フロントエンド**: React 18 + TypeScript
- **地図ライブラリ**: Leaflet + React-Leaflet
- **スタイリング**: Tailwind CSS
- **ビルドツール**: Vite
- **状態管理**: React Hooks (useState, useEffect, useMemo)
- **データ形式**: JSON

## 🚀 セットアップ手順

### 前提条件
- Node.js (v16以上)
- npm または yarn

### インストール

1. リポジトリをクローン
```bash
git clone <repository-url>
cd tokyo-shiritsu-chugaku-map
```

2. 依存関係をインストール
```bash
npm install
```

3. 開発サーバーを起動
```bash
npm run dev
```

4. ブラウザで `http://localhost:5173` にアクセス

### その他のコマンド

```bash
# プロダクションビルド
npm run build

# ビルド結果をプレビュー
npm run preview

# ESLintでコードチェック
npm run lint
```

## 🎨 デザイン仕様

- **カラーパレット**: 青系をベースカラーとした清潔で現代的なデザイン
- **レイアウト**: 地図が主役となる左右分割レイアウト
- **マーカー**: 学校種別（共学/男子校/女子校）ごとに色分け
- **レスポンシブ**: モバイルファーストデザインで全デバイス対応

## 🔧 技術的特徴

- **TypeScript**: 型安全性を確保し、開発効率を向上
- **Component設計**: 再利用可能なコンポーネント設計
- **パフォーマンス**: useMemoを使用した効率的な再レンダリング制御
- **エラーハンドリング**: ErrorBoundaryによる適切なエラー処理
- **アクセシビリティ**: セマンティックHTMLとARIA属性の適切な使用
